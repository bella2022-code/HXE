# HXE Color Tool

HXE is a browser-based color utility for converting color formats, previewing palettes, checking text readability, and exploring curated color systems.

## Website

Open `index.html` directly or deploy the folder as a static site.

## Chrome Extension Prototype

The `chrome-extension` folder contains a Manifest V3 popup prototype with:

- Eyedropper color picking
- Current tab screenshot palette extraction
- Click-to-copy swatches
- Short in-popup usage guidance

To test it in Chrome, open `chrome://extensions`, enable Developer mode, and load the `chrome-extension` folder as an unpacked extension.
